# Examples

## Purpose

`examples/` contains runnable news examples, SDK/API usage examples, and sample non-secret configs.

## Owner

The `sdkwork-news` repository owns examples for news consumers and integration tests.

## Allowed Content

- SDK usage examples.
- API request examples.
- Runnable sample apps or integration examples.

## Forbidden Content

- Production runtime state.
- Real credentials, tokens, or customer data.
- Generated SDK transport output.

## Related Specs

- `../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md`
- `../sdkwork-specs/API_SPEC.md`
- `../sdkwork-specs/SDK_SPEC.md`

## Verification

Run the example-specific command documented beside any added example.
